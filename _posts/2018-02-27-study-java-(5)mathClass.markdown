## Math클래스

### Math클래스는 수학계산을 위한 클래스

- Math클래스는 생성자가 private로 되어 있어 new 연산자를 이용하여 객체 생성 불가

- 모든 메소드와 속성이 static으로 정의되어 객체를 생성하지 않고 사용 가능
